Chapter 3: Forces and Newton's Laws
A force is a push or a pull that can change the motion of an object. Forces can cause an object to start moving, stop moving, speed up, slow down, or change direction. In everyday life, forces occur whenever objects interact with one another. For example, when you push a box across the floor, pull a rope, or when gravity pulls an object toward Earth, forces are involved.
